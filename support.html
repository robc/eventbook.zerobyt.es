Have any questions about Eventbook? If you have, we encourage you to take a look at our Frequently Asked Questions page, and see if we've already answered it. In the event we haven't, then please feel free to contact us with your questions, and we will be happy to investigate.

If you have run into a bug with Eventbook, please contact us so we can investigate. If possible, please include as much detail (or instructions to reproduce as possible), including any details about the event in question. If the event is public, please include the link to the page, as this will help assist us in tracking down any issues.
